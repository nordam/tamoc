tamoc.dbm_utilities¶

Discrete Bubble Model Utilities¶

This module contains utilities to create discrete bubble model (dbm) objects that are especially relevant to blowouts. Among its various capabilities, this module provides an interface to the Oil Library distributed by NOAA ORR.
